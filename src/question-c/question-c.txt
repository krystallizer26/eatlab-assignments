C. You have been tasked with designing a database schema to track student enrollment 
in courses at a university. Each course has a unique ID, name and description. 
Each student has a unique ID, a name, an email address, and a graduation year. 
A student can be enrolled in multiple courses, and a course can have multiple students enrolled. 
Each enrollment should also track the enrollment date.

Design a SQL schema for this problem, including the necessary tables, columns, and relationships.
